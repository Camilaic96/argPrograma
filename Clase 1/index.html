Primera clase

Navegador

Un navegador nos permite el acceso a la web y es el encargado de interpretar las páginas y mostrar el contenido

Frontend

El desarrollo Frontend es la generación de todo el código que interpreta el navegador y con la cual interactúa directamente el internauta

HTML

Es una estándar a cargo del World Wide Web Consortium (W3C), organización dedicada a la estandarización de casi todas las tecnologías ligadas a la web.
Define una estructura básica y un código (denominado código HTML) para la definición de contenido de una página web, como texto, imágenes, videos, juegos, entre otros
